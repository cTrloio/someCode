stop(clearQueue,gotoEnd) clearQueue:如果设置成true，则清空队列。可以立即结束动画。 gotoEnd:让当前正在执行的动画立即完成，并且重设show和hide的原始样式，调用回调函数等。 dequeue(fx) 终止一个自定义的队列函数(移除当前执行的动画) finish() 清除队列(移除所有的动画) delay(duration，queueName) duration:延时时间，单位：毫秒 queueName:队列名词，默认是Fx，动画队列。 jQuery.fx.interval() 设置动画的显示帧速。//运动的时间 run stop dequeue finish
