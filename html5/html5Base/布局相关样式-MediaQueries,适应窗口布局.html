布局相关样式-MediaQueries,响应式口布局

使用方法
@media设备类型 and (设备特征) { 样式代码 }
设备类型
all:所有 screen:电脑 print:打印纸 tv:电视机 ...
设备说明
width （min-width,max-width）浏览器窗口宽度
heigth （min-heigth,max-heigth）浏览器窗口高度
device-width(允许带min-max前缀)设备屏幕分辨值
device-heigth(允许带min-max前缀)设备屏幕分辨率最高值
orientation(不允许带min-max前缀，取值：padding||landscape)浏览器纵向或横向,浏览器高度>=浏览器宽度，取值为padding aspect-ratio(允许带min,max前缀，取值：比例值)窗口的纵横比例值 == 宽度 / 高度。
